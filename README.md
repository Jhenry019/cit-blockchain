# cit-blockchain

### Set up for Windows
- python3 -m venv venv
- venv\Scripts\activate.bat
- cd citblock
- pip install -r requirements.txt
- python3 manage.py migrate

### Set up for MacOS
- python3 -m venv venv
- . venv/bin/activate
- cd citblock
- pip install -r requirements.txt
- python3 manage.py migrate
